# IBPS Clerk Study Planner

A Flutter application designed specifically for college students (like you in your 2nd year, 3rd semester) to plan, track, and prepare for the IBPS Bank Clerk exam alongside their regular degree studies.

## Features

- **Long-term Study Plan:** A structured timeline dividing your preparation into semesters (3rd to 6th), ensuring steady progress without overwhelming your college coursework.
- **Syllabus Tracker:** A detailed breakdown of the IBPS Clerk syllabus (Reasoning, Quantitative Aptitude, English, and General Awareness) with progress indicators to keep track of topics you've covered.
- **Daily Tasks:** A daily checklist to help you build consistent habits, such as reading newspapers, solving a few aptitude questions, and practicing English.

## Tech Stack
- **Flutter & Dart:** Built with the Flutter framework for a smooth, cross-platform experience.
- **Material 3:** Modern UI design with bottom navigation and responsive layouts.

## Getting Started

1. Ensure you have Flutter installed on your system.
2. Clone the repository and navigate to the project directory.
3. Run `flutter pub get` to fetch dependencies.
4. Run the app using `flutter run` on your preferred device or emulator.
